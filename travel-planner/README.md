# Moon Family Travel

GitHub Pages 업로드용 폴더 구조입니다.

## 구조

```text
Repository
├── index.html
└── trips/
    └── jeju-family/
        └── index.html
```

## 업로드 방법

1. GitHub repository 이름 예: `travel-planner`
2. 이 폴더 안의 `index.html` 파일을 repository 최상단에 업로드
3. `trips/jeju-family/index.html` 경로로 제주 여행 페이지 업로드
4. Settings → Pages → Deploy from a branch → `main` / `/root` 선택

## 접속 주소 예시

- 메인: `https://jiyunmoon000.github.io/travel-planner/`
- 제주: `https://jiyunmoon000.github.io/travel-planner/trips/jeju-family/`
